# VisBridge (视界之桥)

> 数据连接视界，可视化搭建认知的桥梁。

**VisBridge** 是一个面向学生的数据可视化大屏项目，旨在帮助初学者从 0 到 1 系统地学习如何构建一个完整的数据大屏应用。项目涵盖数据获取、数据处理、图表渲染、大屏布局、交互联动等核心环节，适合数据可视化入门学习和课程实践。

## 项目特点

- **从零开始**：循序渐进，从项目初始化到完整大屏交付
- **实战导向**：模拟真实业务场景，包含多种常用图表类型
- **模块化设计**：清晰的代码结构，便于学习和扩展
- **技术栈主流**：基于现代前端技术栈构建

## 技术栈

- **前端框架**：React / Vue（待定）
- **图表库**：ECharts / AntV
- **样式方案**：CSS Modules / Less / Tailwind CSS
- **构建工具**：Vite / Webpack
- **数据模拟**：Mock.js / JSON Server

## 快速开始

```bash
# 克隆项目
git clone https://github.com/{username}/VisBridge.git

# 进入项目目录
cd VisBridge

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 学习路线

1. **环境搭建** — 项目初始化、依赖安装、开发环境配置
2. **布局设计** — 大屏分辨率适配、Flex/Grid 弹性布局
3. **数据处理** — 数据模拟、数据清洗、数据格式转换
4. **图表开发** — 柱状图、折线图、饼图、地图等常见图表
5. **交互联动** — 图表间联动、数据下钻、定时刷新
6. **性能优化** — 大屏渲染优化、数据请求优化
7. **部署上线** — 打包构建、静态资源部署
